# Roadmap

## 短期

- テンプレート文面の細部調整を続ける
- warning の出し分けをより自然にする
- サンプル URL と入力補助の改善
- UI 文言の整理

## 中期

- Threads / Instagram / TikTok の表示名取得精度を上げる
- テンプレート種類を増やす
- 媒体別ルールの設定を整理する
- 手動編集しやすい確認フローを強化する

## 長期

- 表示名取得のブラウザ補助方式を検討する
- テンプレート管理をデータ駆動に寄せる
- 複数媒体・複数案件を扱いやすいワークフローへ広げる
- 運用メモやテンプレ仕様をアプリ内に還元する
